## 使用说明
依赖于nonebot2，nonebot2相关信息可以参考官网：https://v2.nonebot.dev/
### 安装方法
安装命令
```
nb plugin install nanobot-plugin-splatoon2tools
```
在nonebot2代码根目录中bot.py
```
nonebot.load_plugin("splatoon_tools")
```
启动bot.py即可
### 命令使用说明
私聊或者群聊发送
```
/工
/凃地
/单排
/组排
```
